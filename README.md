# Anonymous | Question Box

## Stacks
- NEXT.JS
- TAILWINDCSS
- TAILWINDUI
- HEADLESSUI
- MYSQL

## Quick Start

Create `.env` file and put these variables:
```bash
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=

NEXTAUTH_URL=
NEXTAUTH_SECRET=

GITHUB_ID=
GITHUB_SECRET=
```

```bash
npm i
```

Access to /api/setup to setup the database.
